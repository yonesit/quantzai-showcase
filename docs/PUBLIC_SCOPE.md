# Öffentlicher Umfang

Dieses Repository ist ausschließlich eine öffentliche Projektpräsentation von QuantzAI. Es soll GitHub-Besuchern, Suchmaschinen und Interessierten einen professionellen Überblick geben, ohne vertrauliche Projektbestandteile zu veröffentlichen.

## Öffentlich

Öffentlich sind:

- Projektbeschreibung
- allgemeine Architektur
- verwendete Technologien
- Video-Demonstrationen
- ausgewählte Screenshots
- allgemeine Entwicklungsinformationen

## Privat

Privat bleiben:

- Quellcode
- Strategielogik
- Modelle
- Parameter
- Konfigurationen
- Zugangsdaten
- interne Daten
- produktive Infrastruktur
- nicht öffentliche Dokumentation

## Grenzen der Darstellung

QuantzAI wird nicht als fertiges profitables Handelssystem dargestellt. Dieses Repository enthält keinen ausführbaren Tradingbot, keine Handelsstrategie und keine vertraulichen technischen oder finanziellen Kennwerte.

QuantzAI ist ein privates Forschungs- und Entwicklungsprojekt. Die öffentliche Darstellung ist keine Finanzberatung und enthält kein Gewinnversprechen.
