## Projektüberblick

QuantzAI ist ein privat entwickeltes Forschungs- und Softwareprojekt von Yones Alizai. Das Projekt beschäftigt sich mit der datenbasierten Analyse von Forex- und Rohstoffmärkten und verbindet Marktdatenverarbeitung, Machine Learning, Backtesting, Paper-Trading und eine grafische Desktop-Oberfläche.

Dieses Repository dient ausschließlich als öffentliche Projektpräsentation. Es enthält keine interne Strategielogik, keine Modelle, keine vertraulichen Konfigurationen und keinen vollständigen Quellcode des privaten Projekts.

## Ziele des Projekts

- strukturierte Analyse historischer Marktdaten
- reproduzierbare Datenverarbeitung und Feature Engineering
- zeitbasierte Validierung von Machine-Learning-Modellen
- Bewertung von Modellverhalten unter realistischeren Annahmen
- Integration von Risikomanagement, Paper-Trading und Audit-Logging
- verständliche Bedienung über eine Desktop-Anwendung

## Kernbereiche

| Bereich | Öffentliche Beschreibung |
| --- | --- |
| Marktdaten | Verarbeitung historischer Forex- und Rohstoffdaten |
| Feature Engineering | Aufbereitung datenbasierter Merkmale für Modelltraining und Bewertung |
| Machine Learning | Modellierung und Validierung mit zeitlicher Trennung der Daten |
| Backtesting | Prüfung von Modellentscheidungen anhand historischer Daten |
| Risikomanagement | Komponenten zur kontrollierten Bewertung und Begrenzung von Szenarien |
| Paper-Trading | Simulierte Ausführung ohne produktiven Kapitaleinsatz |
| Desktop-Oberfläche | Grafische Bedienung, Monitoring und Audit-Logging |
| Qualitätssicherung | Automatisierte Tests und nachvollziehbare Protokollierung |

## Technologien

| Kategorie | Technologien |
| --- | --- |
| Programmiersprache | Python 3.11 |
| Machine Learning | LightGBM, scikit-learn |
| Datenverarbeitung | pandas, PyArrow |
| Desktop-Oberfläche | PySide6 |
| Trading-Anbindung | MetaTrader 5 |
| Speicherung | SQLite |
| Logging und Tests | Loguru, pytest |
