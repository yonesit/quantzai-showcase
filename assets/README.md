# Assets

In `assets/screenshots` können später öffentliche und vorab geprüfte Screenshots für die Projektpräsentation abgelegt werden.

Geeignete Dateinamen wären zum Beispiel:

- `quantzai-dashboard.png`
- `quantzai-model-view.png`
- `quantzai-risk-view.png`
- `quantzai-paper-trading.png`

Vor der Veröffentlichung müssen Screenshots auf private Informationen, Zugangsdaten, interne Pfade, vertrauliche Kennzahlen und nicht öffentliche Projektdetails geprüft werden.
